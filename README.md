# Robotic-Task-FSM

一个轻量、高扩展性的机器人运动控制有限状态机框架，支持事件驱动、状态回调与线程安全机制，适用于机器人路径规划、避障执行等复杂任务调度流程。

## 特点

* **模块化设计** ：FSM 封装在Fsm 类中，状态逻辑清晰可控。
* **自定义状态转换表**：可根据自己的任务需求定义相应的状态转换表
* **状态/事件枚举定义** ：支持强类型的 `SystemState` 和 `SystemEvent`。
* **事件驱动机制** ：状态转移由事件触发，便于任务驱动控制。
* **灵活的回调注册** ：支持为每个状态或事件设置自定义行为。
* **线程安全可选** ：可通过内部互斥锁保护多线程访问（`getOptionalLock()`）。
* **可视化调试接口** ：日志输出 `logMessage`，便于分析状态跳转过程。
